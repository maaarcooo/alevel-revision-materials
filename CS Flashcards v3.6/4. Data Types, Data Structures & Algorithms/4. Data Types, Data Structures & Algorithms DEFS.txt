How does a stack differ from a queue? | A stack is LIFO (last in, first out) while a queue is FIFO (first in, first out)
How does breadth first traversal differ from depth first traversal? | Breadth first visits all neighbours of a node before moving to the next level. Depth first travels as far as possible along one route before backtracking
How does a full adder differ from a half adder? | A half adder adds two single bits. A full adder combines two half adders and also accepts a carry-in bit, handling three inputs total
How does an array differ from a tuple? | An array stores data of the same type and can be modified. A tuple can store different data types but is immutable (cannot be changed once created)
