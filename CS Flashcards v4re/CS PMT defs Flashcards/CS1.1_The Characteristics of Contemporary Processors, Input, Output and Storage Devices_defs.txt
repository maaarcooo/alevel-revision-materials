What is the Data Bus? | A bi-directional bus for carrying data and instructions between the processor and memory
A bi-directional bus that carries data and instructions between the processor and memory — what is it called? | The Data Bus
How does the Address Bus differ from the Data Bus? | The Address Bus is unidirectional and carries memory addresses only. The Data Bus is bi-directional and carries actual data and instructions.
What are the three stages of the Fetch-Decode-Execute (FDE) cycle? | Fetch (supply address, retrieve instruction from memory), Decode (interpret instruction, read required data), Execute (CPU carries out the required actions)
What happens during the Fetch stage of the FDE cycle? | The address is supplied and the instruction is retrieved from memory
What happens during the Decode stage of the FDE cycle? | The instruction is interpreted, and the required data is read and retrieved from their addresses
What happens during the Execute stage of the FDE cycle? | The CPU carries out the required actions specified by the instruction
What is Harvard Architecture? | A computer architecture that stores data and instructions in separate memories, allowing the next instruction to be read while data is currently being read or written
A computer architecture using separate memory for data and instructions so they can be accessed simultaneously — what is it called? | Harvard Architecture
What advantage does Harvard Architecture gain from using separate memories for data and instructions? | The next instruction can be fetched while data is still being read or written, increasing throughput
What is the Memory Address Register (MAR)? | A special register that stores the memory address of the next instruction to load or data to use

What is the Memory Data Register (MDR)? | A special register that temporarily stores data to be read from or written to the computer's memory
A register that temporarily holds data being transferred to or from main memory — what is it called? | The Memory Data Register (MDR)
How does the MAR differ from the MDR? | The MAR holds the memory address being accessed. The MDR holds the actual data being read from or written to that address.

What is Von Neumann Architecture? | A computer architecture where a single control unit manages program control via a linear sequence of Fetch-Decode-Execute cycles
A computer architecture using a single control unit and a linear FDE cycle sequence — what is it called? | Von Neumann Architecture
How does Von Neumann Architecture differ from Harvard Architecture? | Von Neumann stores data and instructions in the same memory and processes them sequentially. Harvard uses separate memories for data and instructions, allowing simultaneous access.
What is a Complex Instruction Set Computer (CISC)? | A processor design that can execute a series of tasks in a single complex instruction built into the hardware, using less RAM but preventing pipelining
A processor design using complex, multi-task instructions built into hardware — what is it called? | CISC (Complex Instruction Set Computer)
What is a Reduced Instruction Set Computer (RISC)? | A simpler processor design that can only execute a single simple instruction each clock cycle, using more RAM but allowing pipelining
A processor design using simple single-cycle instructions that supports pipelining — what is it called? | RISC (Reduced Instruction Set Computer)
How does CISC differ from RISC? | CISC uses complex multi-task instructions (less RAM, no pipelining). RISC uses simple single-cycle instructions (more RAM, supports pipelining).
Why can RISC processors use pipelining but CISC processors cannot? | RISC instructions each take a single clock cycle and are uniform in length, making it easy to overlap stages. CISC instructions vary in complexity and length, making pipelining impractical.
Why does a CISC processor use less RAM than a RISC processor? | CISC can encode a series of tasks into a single complex instruction, so fewer instructions are needed to complete a program, reducing memory usage
What is a Graphic Processing Unit (GPU)? | A specialised processing unit with a huge number of small cores that allow efficient parallel computation for tasks such as graphics, machine learning, and data mining
A processing unit with many small cores designed for efficient parallel computation — what is it called? | A GPU (Graphic Processing Unit)
